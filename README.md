# Sapphire-Qwen3-TTS
Local AI text-to-speech plugin for Sapphire with voice cloning, voice design, and 9 preset speakers. Uses Qwen3-TTS models from Alibaba/Qwen. Design voices from natural language descriptions, clone from 3-second audio clips, or customize built-in speakers with style instructions. Fully self-contained — no core Sapphire changes needed.
